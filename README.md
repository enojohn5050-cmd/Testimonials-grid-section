# Testimonials Grid Section

This is my solution to the Frontend Mentor Testimonials Grid Section challenge.

## Overview

The project displays a responsive testimonials grid using HTML and CSS. The layout changes to fit different screen sizes, providing a good user experience on both desktop and mobile devices.

## Built With

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Mobile-first responsive design

## What I Learned

While building this project, I learned how to:

- Build responsive layouts using CSS Grid.
- Use Flexbox to align content.
- Organize CSS using variables.
- Position grid items with `grid-column` and `grid-row`.
- Create responsive designs using media queries.
- Push projects to GitHub and deploy them with GitHub Pages.
